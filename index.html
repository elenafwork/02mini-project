CSS Snippet Cheatsheet

Flexbox - Reverse

Reverse positions of the elements in a row

  .reverse { 
                     display: flex; 
                    flex-direction: row-reverse;  
                 }

Font-variant - Small Caps

Small-caps displays text in small capital letters

small-caps::first-line {
            font-variant: small-caps;
            font-size: 120%;
            }

Perspective

Perspective gives an element perspective in 3D.

.perspective {
                    perspective: 500px;
                    }
                .card {
                  transition: transform 1s ease;
                  &:hover {
                    transform: rotateY(-60deg);
                      }
                }

Media Query Screen

Transforms the elements properties depending om screen size 

 @media screen and (max-width: 992px) {
                    flex-direction: column;
                    background-color: red
                }

Flexbox: wrap-reverse

Multi-lines, opposite to direction defined by flex-direction 

 .container  {
                    display: flex;
                    flex-wrap: wrap-reverse;
            
                    }

Pseudo-classes

A keyword added to a selector that specifies a special state of selected element(s) 

                button:hover
                    transform: scale(110%)
